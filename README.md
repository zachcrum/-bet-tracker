# Live SGM Bet Tracker

Mobile-first same-game multi tracker for the Knicks @ Spurs live game.

## GitHub Pages

To publish it:

1. Go to **Settings → Pages**.
2. Source: **Deploy from a branch**.
3. Branch: **main**.
4. Folder: **/root**.
5. Save.

Your live page should be:

`https://zachcrum.github.io/-bet-tracker/`

The tracker is a standalone `index.html` file with no build step.
